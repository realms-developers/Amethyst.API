# Amethyst API
Rich, modern and optimized API for Terraria Server.

## Installation

Firstly, clone the repository from GitHub:
```
git clone https://github.com/realms-developers/Amethyst.API -r
```

Launch server:
### Windows
```
start.bat <arguments>
```
### Linux
```
./start.sh <arguments>
```
